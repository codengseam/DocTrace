# 睡眠与精力修复课 · 写作指南（最终版）

> 本文件是所有生成 agent 必读的写作规范。请先完整阅读本文件，再开始写章节。

## 一、专栏定位
融合传统养生智慧与现代睡眠科学的精力修复指南。面向普通人/学生/高压上班族。讲透睡眠、休息、冥想、呼吸、起居调摄的底层逻辑，给出零门槛可落地方法。属"子"类（医家养生）。

## 二、硬约束（P0，违反必须重写）
1. **不编造**：不编造来源、不编造名家/专家点评、不编造实验数据与数字。记不清的史实/出处/评语宁可不写，不可硬写。
2. **权威来源白名单**（只能用以下来源）：
   - 传统经典：《黄帝内经》（仅《上古天真论》《四气调神大论》《营卫生会》等起居/养神/睡眠/四时篇章）、《千金要方》（养性/睡眠/静息）、《本草纲目》（常见食药助眠）、《抱朴子》（呼吸静养）。禁用《难经》《神农本草经》。
   - 现代中医专家：吴中朝（睡眠养生）、夏萌（临床营养）、徐文兵（《内经》通俗解读）、佟彤（疲劳气血）。禁用米晶子。
   - 现代睡眠科学：Matthew Walker《我们为什么要睡觉》、Nick Littlehales《睡眠》（R90法）、西野精治《斯坦福好睡眠》（黄金90分钟）、中国睡眠研究会《中国成人睡眠卫生指南》、AASM临床指南、Jon Kabat-Zinn（MBSR/《正念》）、Andrew Weil（4-7-8呼吸）。补充：Shawn Stevenson、Loehr&Schwartz《精力管理》、久贺谷亮《高效休息法》、Mark Williams《正念》。
3. **健康安全边界**：严重失眠（持续>1个月）、伴心悸/呼吸暂停/日间严重功能障碍者，必须明确引导就医。文末加免责声明："本课为生活方式建议，不替代医疗诊断，严重睡眠问题请就医。"
4. **传统养生标注**：所有传统养生方法必须标注"（传统养生参考）"，与医学建议区分。

## 三、语言风格
- 中文为主，自然口语化，像讲故事/硬核科普博主，有节奏有温度。
- 叙事生动、分析冷静、悟道深刻。有场景、有人物处境、有转折，不写干巴巴说明文。
- **禁止AI套路句式**："我们可以看到""这告诉我们""综上所述""不是X而是Y""他不是X，是Y"等。
- **禁止现代学科术语硬套**（如博弈论、底层逻辑、坐标系）。
- 关键概念首次出现用一句话白话解释。
- 专有名词（R90/REM/4-7-8/MBSR等）豁免中英文混杂禁令。

## 四、引用规范
- 行内引用≤3处/千字。格式：「原文」——《书·篇》或"据《我们为什么要睡觉》研究结论"。
- 经典原文≤20字/段，必须配白话解读。
- 大意转述须标"大意据《XXX》"。
- 删除所有"（见上文）""（详见下章）"等跳转提示。
- 文末统一列"## 参考来源"，每条注明书名/篇名及核心结论。

## 五、单章结构（核心方法章用完整7段；开篇/总结/列表型章可简化）
1. **讲清楚**：问题、受众、重要性（开篇章可更自由）
2. **讲原理（讲本质）**：核心生理/养生逻辑、底层规律、跨场景共性
3. **讲实验/案例**：一个经典实验、历史故事或真实案例（须有来源；传统章可用历史故事/经典记载替代现代实验）
4. **讲核心方法**：具体步骤、要点、时长频率
5. **讲权威依据**：相关研究结论、经典原文、专家观点
6. **讲实操落地**：日常用法、常见错误、不同情况调整
7. **结语**：一句话总结本章核心
